# 📂 Estructuras de Datos e Información (EDI)

¡Bienvenido/a! Este repositorio contiene todos los ejercicios, prácticas y algoritmos desarrollados en la asignatura EDI de la carrera de Ingeniería Informática y Telemática. 🎓

### 🚀 Objetivo del Repositorio
El propósito de este espacio es doble:
1. **Historial Personal:** Guardar mi progreso y evolución en el dominio de Java. 📈
2. **Recurso Público:** Servir de apoyo y consulta para otros compañeros que estén cursando la asignatura o quieran aprender estructuras de datos. 🤝

### 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Java ☕
* **Paradigma:** Programación Orientada a Objetos (POO)
* **Entorno:** [BlueJ / IntelliJ / VS Code]

### 📚 Contenido Destacado
Aquí encontrarás implementaciones de:
* 🔹 **TDA Lineales:** Pilas, Colas y Listas Enlazadas.
* 🔹 **TDA No Lineales:** Árboles, herencia, composicion, polimorfismo...
* 🔹 **Algoritmos:** Búsqueda, ordenación y gestión de memoria.
* 🔹 **Prácticas:** Resoluciones de problemas planteados en clase.
